# JavaBackend
Data Migration From MongoDb To Excel
I have attached 3 java Programs 
1. one.java : To establish the connection with MongoDb and retrieve the tables(collection) from MongoDB and dsiplay the list of tables to the user, from which the    user will select one table to migrate
2.Fieldnm.java : to get the Field names(attributes) of the selected table and its values and store them in arrayList,Also check the missing fields and values and rebuild the missing fields.
3.CreateCsv.java : to store the data in the Csv format that can be opened in Excel.
